## 下载

### Docker

```
docker pull apachecn0/apachecn-android-zh
docker run -tid -p <port>:80 apachecn0/apachecn-android-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g apachecn-android-zh
apachecn-android-zh <port>
# 访问 http://localhost:{port} 查看文档
```
